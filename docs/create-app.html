Create a claim-bot GitHub App

This pre-fills a GitHub App with exactly the permissions claim-bot needs (Issues + Pull requests read/write, organization Projects read/write, no webhook). Enter the account that owns the board and the repo, then click create.


GitHub opens its App-creation page with everything filled in. Click Create GitHub App, then: generate a private key, copy the App ID, and install the App on the repo that holds your issues. App names are globally unique, so adjust the suggested name if it's taken.

After creating

  1. On the new App's page, Generate a private key (downloads a .pem).
  2. Note the App ID.
  3. Install App on the repository that holds your issues.
  4. In that repo → Settings → Secrets and variables → Actions: add variable CLAIM_BOT_APP_ID and secret CLAIM_BOT_APP_PRIVATE_KEY.
